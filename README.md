# 🍰 Cantinho Doce da Mimi

Sistema de pedidos online desenvolvido para um pequeno negócio real.

🔗 **Acesse o site:**  
https://cantinho-doce-mimi.netlify.app/

## 🚀 Funcionalidades
- Envio de pedidos de bolos de pote
- Cálculo automático de valores
- Integração front-end e back-end
- Persistência de dados em PostgreSQL
- Sessão por cliente
- Deploy em produção

## 🛠️ Tecnologias utilizadas
- HTML, CSS, JavaScript
- React
- Node.js + Express
- PostgreSQL
- Git & GitHub
- Netlify (Front-end)
- Render (Back-end e Banco)
